HTML
HTML (HyperText Markup Language) is a popular type of mark-up language classed under the declarative paradigm.It is the industry standard markup language for creating web pages as well as web applications.HTML documents consist of elements known as tags are surrounded by angular brackets (such as the ones seen in the example image to the left). Even though these tags aren’t seen, these tags help define what is and isnt code for the browser to interpret.

CSS
CSS stands for Cascading Style Sheets. It is a stylesheet language, utilized mainly to describe how an HTML document is to be presented. CSS enables separation of presentation and content. PHP
HyperText Processor is a general purpose programming language designed for web development. PHP is an imperative type programming language, meaning that they can be used to define functions and describe specific tasks. PHP code may be executed through a command line interface or may be embedded within an html document. This can be done by using the already provided tags. PHP today is incorporated as the industry standard in developing web applications. In our case, we used PHP to obtain data from the topuniversities.com to show the current ranking of the University of Manchester without having to leave the project website.
